## 2026-03-24 - [Avoid InjektionException when extracting UseCases] **Learning:** [Kohesive Injekt is a service locator that requires explicit registration in a DI module (e.g., an `AppModule`). Injecting new unregistered UseCases via `private val mergeMangaUseCases: MergeMangaUseCases = Injekt.get()` causes runtime crashes.] **Action:** [When creating new UseCases, instantiate them directly (e.g., `private val mergeMangaUseCases = MergeMangaUseCases()`) and rely on the default `Injekt.get()` parameters inside their constructors to resolve framework dependencies.]
