## 2024-05-18 - [State Hook Extraction] **Learning:** Extracting complex `remember` logic into a dedicated custom state hook `@Composable` (e.g., `rememberThemeColorState`) is a highly effective pattern to reduce UI component clutter in Jetpack Compose, even when the target isn't explicitly a visual node. **Action:** Next time an overly complex `val = remember(...) { ... }` block is encountered, extract it into a dedicated, focused composable function returning the state, keeping the parent UI code readable and declarative.
