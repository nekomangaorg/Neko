## 2026-03-15 - [Avoid Intermediate Lists with mapNotNull] **Learning:** Using `.filter {}.map {}` chains on large lists allocates an intermediate list, which increases GC overhead and reduces performance on critical paths like resolving the next chapter. **Action:** Replace `.filter {}.map {}` chains with `.mapNotNull { if (...) ... else null }` to avoid intermediate list allocations and improve performance, especially when quickly jumping to reading.
