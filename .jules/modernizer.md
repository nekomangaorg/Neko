## 2024-05-24 - [Intent.ACTION_MEDIA_SCANNER_SCAN_FILE] **Learning:** Intent.ACTION_MEDIA_SCANNER_SCAN_FILE is deprecated in Android Q and above. Use MediaScannerConnection.scanFile instead. The path given to scanFile must be an absolute String path. If you only have a Uri, use uri.path, or if possible refactor to file.absolutePath. **Action:** Refactored scanMedia method to use MediaScannerConnection.scanFile and passed appropriate string paths.
