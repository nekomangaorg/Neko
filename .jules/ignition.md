## 2024-05-17 - Defer CookieManager Initialization **Learning:** `CookieManager.getInstance()` performs synchronous file I/O on the main thread and initializes the Chromium WebView engine, blocking Time to Initial Display (TTID). **Action:** Defer initialization to a background coroutine via `ProcessLifecycleOwner.get().lifecycleScope.launch(Dispatchers.Default)` and wrap any UI fallback (e.g., Toasts) with `withContext(Dispatchers.Main)` to avoid `CalledFromWrongThreadException`.
