## 2024-05-19 - [Lazy Injection for TTID] **Learning:** Eager instantiation of singleton dependencies in `AppModule.kt` using `addSingleton()` can cause a significant bottleneck during the main thread `onCreate()` due to cascaded synchronous object creations, especially those doing disk I/O (like `DelayedTrackingStore` via SharedPreferences). **Action:** Always use `addSingletonFactory { }` to defer object creation until its first real usage, improving cold start speed.
## 2024-05-19 - [Background Setup of CookieManager] **Learning:** `CookieManager.getInstance()` performs synchronous disk IO which blocks the main thread when executed directly inside `Application.onCreate()`. **Action:** Move it to a background thread (`Dispatchers.Default`) using Jetpack App Startup, ensuring that UI updates like Toasts triggered by potential failures are routed back to `Dispatchers.Main`.
