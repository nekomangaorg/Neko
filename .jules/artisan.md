## 2025-02-23 - [Talkback] **Learning:** When creating complex Jetpack Compose layouts like `MangaRow`, missing content descriptions can cause TalkBack to miss important state context (unread/download badges, statuses). By using `.semantics(mergeDescendants = true) { this.contentDescription = ... }` on the parent and dynamically constructing the content description string within a `remember` block, we can group information natively for screen readers without breaking interaction paths. **Action:** Next time when fixing accessibility on a composite row item, always review if grouping (`mergeDescendants`) is necessary and aggregate all contextual state into a single cohesive reading label.
