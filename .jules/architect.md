## 2024-04-01 - [Move Stats UI logic to presentation package] **Learning:** [In the Nekomanga project, Jetpack Compose ViewModels and UI helpers belong in their respective feature packages under `org.nekomanga.presentation.screens.<feature>` rather than legacy `eu.kanade.tachiyomi.ui.*` packages] **Action:** [When encountering UI components or ViewModels for Compose screens in the legacy `eu.kanade.tachiyomi` hierarchy, relocate them to the appropriate `org.nekomanga.presentation.screens` package to maintain cohesive Clean Architecture structure]
